# Assumptions

- "AWS" is treated as Amazon marketplace for this run. If it means Amazon Web Services, the product surface and target user need to change.
- The first real segment should be cross-border Amazon sellers, not all Taobao, JD, and cross-border sellers.
- Sellers care about compliance and speed enough to pay for a reviewed SKU image pack.
- GPT Image 2 can generate useful variants while preserving product fidelity, but every output needs human review before delivery.
- Taobao and JD official image rules still need manual verification from current seller-console or official rule pages.
- Subscription should wait until repeat SKU usage is proven.
