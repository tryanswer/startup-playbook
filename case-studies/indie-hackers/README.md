# Indie Hackers Case Patterns

This directory stores pattern summaries mined from Indie Hackers founder stories. Full article HTML and extracted text should stay in ignored local tool output, not committed here.

## Intended Output

- repeated validation moves;
- first acquisition channels;
- pricing and revenue signals;
- anti-patterns and caveats;
- source-linked evidence rows;
- updates to `skills/founder-case-patterns`.
- staged guide summaries such as `starting-up-guide.md`;
- updates to `skills/indie-hackers-starting-up`.

## Current Snapshot

- `patterns.md`: generated from 499 Indie Hackers stories; 498 pages were readable and 1 historical URL returned 404.
- `starting-up-guide.md`: generated from the Starting Up hub plus 80 indexed resources; 78 pages were readable and 2 Twitter/X links were blocked by fetch.
- Raw HTML and extracted article text are intentionally kept in ignored `tools/case-study-skill-miner/output/`.
