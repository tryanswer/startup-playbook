# Indie Hackers Case Patterns

This directory stores pattern summaries mined from Indie Hackers founder stories. Full article HTML and extracted text should stay in ignored local tool output, not committed here.

## Intended Output

- repeated validation moves;
- first acquisition channels;
- pricing and revenue signals;
- anti-patterns and caveats;
- source-linked evidence rows;
- updates to `skills/founder-case-patterns`.
